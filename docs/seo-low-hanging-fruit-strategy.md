# Low-Hanging-Fruit SEO Strategy — SopKit

## Goal

Create fast-indexing, low-keyword-difficulty landing pages for long-tail tool intent terms. SopKit targets 405+ free online tools across 12 categories with a focus on privacy-first, browser-based processing.

## Strategy

1. **Target long-tail query patterns** per tool:
   - `free <tool name> online`
   - `<tool name> without signup`  
   - `<tool name> for <niche use-case>`
   - `<tool name> under <size/limit> KB`

2. **On-page optimization**: Keep title tags and H1 tightly aligned with search intent. Include primary keyword in first 100 words.

3. **Internal linking**: Link from category hubs and related-tool blocks. Prioritize exact-use pages (e.g., Compress to 20KB) from generic parent pages (e.g., Image Compressor).

4. **Technical SEO**: Every tool has a canonical URL, XML sitemap entry, robots.txt allowance, and JSON-LD structured data (SoftwareApplication, FAQPage, HowTo, BreadcrumbList).

5. **Content depth**: Expand with unique FAQs, HowTo guides, and feature lists. Target 800-1500 words per tool page.

6. **AI search optimization**: Maintain `/llms.txt`, allow AI crawlers in `robots.txt`, and include citable, factual content.

## Priority clusters

- Exam photo/signature tools (SSC, UPSC, NEET, JEE, PAN, passport)
- KB-specific image compression (10KB, 20KB, 30KB, 50KB, 100KB, 200KB)
- Student calculators (attendance, SGPA, CGPA, required marks)
- API key testers (OpenAI, Claude, Gemini, Groq, Stripe, Twilio)
- Business tools (QR code generators, SEO tools, invoice generators)
